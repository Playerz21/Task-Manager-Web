# Task-Manager-Web

#Intro
This is a React(Javascript) and Django(Python) combined project created by me.

#Requirements

1. node modules in frontend folder
2. react
3. django
4. djang-rest-frame-work
5. babel
6. bootstrap
etc.

#To start the server
1. Open Cmd
2. cd to this file
3. Type python manage.py runserver
4. Open a new CMD then follow step 2
5. cd frontend
6. npm run dev
